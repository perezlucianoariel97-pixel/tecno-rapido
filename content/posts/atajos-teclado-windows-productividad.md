El ratón es cómodo pero lento. Los usuarios avanzados hacen en un segundo lo que a otros les lleva diez. Aquí están los 30 atajos de Windows que más tiempo te van a ahorrar.

## Atajos esenciales (que todo el mundo debería saber)

| Atajo | Qué hace |
|-------|----------|
| `Ctrl + Z` | Deshacer |
| `Ctrl + Y` | Rehacer |
| `Ctrl + C / V / X` | Copiar / Pegar / Cortar |
| `Ctrl + A` | Seleccionar todo |
| `Ctrl + S` | Guardar |
| `Alt + F4` | Cerrar ventana/programa |
| `F2` | Renombrar archivo seleccionado |

## Atajos de ventanas (los más útiles)

| Atajo | Qué hace |
|-------|----------|
| `Windows + D` | Mostrar/ocultar escritorio |
| `Windows + E` | Abrir Explorador de archivos |
| `Windows + L` | Bloquear el PC |
| `Windows + ←/→` | Anclar ventana a mitad de pantalla |
| `Windows + ↑` | Maximizar ventana |
| `Windows + ↓` | Minimizar ventana |
| `Windows + Tab` | Vista de tareas (todas las ventanas abiertas) |
| `Alt + Tab` | Cambiar entre ventanas abiertas |
| `Windows + Número` | Abrir/enfocar programa en la barra de tareas |

## Atajos del navegador (Chrome/Edge/Firefox)

| Atajo | Qué hace |
|-------|----------|
| `Ctrl + T` | Nueva pestaña |
| `Ctrl + W` | Cerrar pestaña actual |
| `Ctrl + Shift + T` | Reabrir última pestaña cerrada |
| `Ctrl + Tab` | Siguiente pestaña |
| `Ctrl + L` | Ir a la barra de direcciones |
| `Ctrl + F` | Buscar en la página |
| `F5 / Ctrl + R` | Recargar página |
| `Ctrl + Shift + N` | Ventana de incógnito |

## Atajos avanzados de Windows

| Atajo | Qué hace |
|-------|----------|
| `Windows + V` | Historial del portapapeles (guarda lo último que copiaste) |
| `Windows + .` | Insertar emoji |
| `Windows + Shift + S` | Captura de pantalla recortada |
| `Ctrl + Shift + Esc` | Administrador de tareas directo |
| `Windows + I` | Abrir Configuración |
| `Windows + X` | Menú de herramientas avanzadas |
| `Windows + P` | Proyectar pantalla (para monitores externos) |
| `Windows + G` | Barra de juegos de Xbox |
| `Windows + K` | Conectar a dispositivos Bluetooth/inalámbricos |

## El atajo más subestimado: Windows + V

La mayoría desconoce que Windows tiene un **historial del portapapeles**. Con `Windows + V` puedes ver todo lo que has copiado en la sesión actual y volver a pegar cualquier elemento anterior. Increíblemente útil.

Para activarlo la primera vez: `Windows + V` → Activar.

## Cómo memorizarlos

No intentes aprenderlos todos de golpe. Elige **3 atajos nuevos por semana** y úsalos conscientemente hasta que sean automáticos. En un mes dominarás los más importantes.
