No necesitas instalar OBS ni ningún programa de terceros para grabar tu pantalla en Windows. El sistema ya incluye varias herramientas nativas que funcionan perfectamente para la mayoría de los casos.

## Método 1: Xbox Game Bar (el más rápido)

La Xbox Game Bar viene preinstalada en Windows 10 y 11. Aunque tiene "Xbox" en el nombre, funciona para grabar cualquier ventana, no solo juegos.

**Cómo usarla:**

1. Abre la aplicación o ventana que quieres grabar
2. Presiona `Windows + G` para abrir la Game Bar
3. Aparecerá un panel flotante con controles
4. Haz clic en el botón **Grabar** (círculo) o presiona `Windows + Alt + R`
5. Para detener la grabación: `Windows + Alt + R` de nuevo
6. El vídeo se guarda en `C:\Users\TuUsuario\Videos\Capturas`

**Atajo directo sin abrir el panel:**
- `Windows + Alt + R` — iniciar/detener grabación
- `Windows + Alt + M` — activar/desactivar micrófono durante la grabación
- `Windows + Alt + G` — guardar los últimos 30 segundos (modo clip)

**Limitación:** solo graba una ventana a la vez, no el escritorio completo.

## Método 2: Snipping Tool (Windows 11)

En Windows 11, la Snipping Tool actualizada incluye grabación de pantalla además de capturas.

1. Busca **"Snipping Tool"** en el menú inicio
2. Haz clic en el icono de la cámara de vídeo (junto al de la cámara de foto)
3. Selecciona el área de la pantalla que quieres grabar
4. Haz clic en **Iniciar**
5. Cuando termines, haz clic en **Detener**
6. El vídeo se guarda automáticamente

**Ventaja sobre la Game Bar:** puedes seleccionar exactamente el área que quieres grabar.

## Método 3: PowerPoint (si tienes Office)

Sí, PowerPoint puede grabar tu pantalla. Es útil para crear tutoriales.

1. Abre PowerPoint
2. Ve a **Insertar → Grabación de pantalla**
3. Selecciona el área a grabar
4. Haz clic en **Grabar**
5. Cuando termines, haz clic en **Detener**
6. El vídeo se inserta en la diapositiva
7. Clic derecho sobre el vídeo → **Guardar archivo multimedia como** para exportarlo

## Método 4: Microsoft Teams o Zoom (para reuniones)

Si quieres grabar una videollamada, tanto Teams como Zoom tienen grabación integrada. En Teams: botón de tres puntos durante la reunión → Iniciar grabación.

## Comparativa de métodos nativos

| Método | Área | Audio | Webcam | Facilidad |
|--------|------|-------|--------|-----------|
| Xbox Game Bar | Una ventana | ✓ | ✗ | Muy fácil |
| Snipping Tool | Área seleccionada | ✗ | ✗ | Muy fácil |
| PowerPoint | Área seleccionada | ✓ | ✓ | Fácil |

## Cuándo sí necesitas OBS

Los métodos nativos son suficientes para uso básico. Necesitas OBS u otro software externo si quieres:

- Grabar múltiples fuentes simultáneamente
- Hacer streaming en directo a Twitch o YouTube
- Añadir overlays, transiciones o efectos
- Grabar durante horas sin interrupciones
- Control avanzado de calidad y bitrate

## Consejos para mejores grabaciones

- **Cierra notificaciones** antes de grabar: activa el modo No molestar (`Windows + A`)
- **Usa auriculares** si grabas con micrófono para evitar el eco
- **Revisa el espacio** en disco: los vídeos pueden ocupar mucho
- **Graba a 1080p** para buena calidad sin archivos enormes
